AJAX Todo List

Switch to english | česky

AJAX Todo List is an example of good MVC based client side application design.

I am not planning for further development of this application right now. This may change in the future.

Features

  • Client side based TODO list with task categories that communicates with server using JSON-RPC 1.1.
  • Clean OOP design based on the MVC pattern.
  • You can use Texy! HTML formating in the note.
  • Trivial installation (copy and mkdir -m 0777 db).
  • Almost completely stylable by CSS.
  • No authentication.
  • KISS server side.

Compatibility

Tested in Safari 3.1.2, Firefox 3, IE 6.0, IE 7.0

Please use real browsers. That is anything but IE.

Installation

  • Copy files to the directory on the server.
  • Create directory db/ writable by the webserver.
  • Enjoy TODOing!
